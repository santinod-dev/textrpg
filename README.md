# Simple Text RPG

A beginner project: a simple text RPG that runs in the terminal.

## How to Run

1. Compile the Java files:
   ```
   javac *.java
   ```
2. Run the program:
   ```
   java Main
   ```

## Description

This is a terminal-based RPG game written as a beginner CS project.

## Features

- Character creation
- Basic combat
- Turn-based system
